## Examples

```julia
m = Vasicek(0.136,0.0168,0.0119)
s = ScenarioGenerator(
        1,                              # timestep
        10,                             # projection horizon
        ESG.Vasicek(0.136,.0168,.0119), # ESG model
        0.01                            # starting rate
    )
```

This can be iterated over, or you can collect all of the rates like:

```julia
rates = collect(s)
```

or 

```julia
for r in s
    # do something with r
end
```

And the package integrates with [Yields.jl](https://github.com/JuliaActuary/Yields.jl) if loaded:

```julia
using Yields

yieldcurve(s)

```
